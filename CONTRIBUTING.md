# 贡献与纠错说明

欢迎对本仓库内容进行纠错和完善。

如果你在阅读过程中发现：

- 内容存在错误
- 推导不清晰
- 表述不严谨
- 代码存在 bug
- 有更好的解释或补充
  
或者：

- 我也想把自己的成果上传

都可以通过以下方式参与改进。

---

## 提交 Issue

1. 打开对应仓库
2. 点击 Issues
3. 新建 Issue
4. 简要说明：
   - 哪个文件
   - 哪一部分有问题
   - 建议修改方式（如有）

---

## 提交 Pull Request（直接修改）

如果你希望直接修正内容：

1. Fork 本仓库
2. 在你的仓库中修改
3. 提交 Pull Request


---
## 方式三：使用 Discussions

欢迎在 Discussions 板块发起话题，共同交流。


感谢每一位愿意参与贡献的同学。
